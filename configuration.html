Command Line Options

-h, --help
Prints help information
-V, --version
Prints version information
-b, --bind <IP>
Specify a custom IP to bind to. Default: 127.0.0.1
-p, --port <PORT>
Specify a custom port. Default: 6363
--pidfile <PIDFILE>
Generate a pidfile at the specified path. Example: /var/run/hanbaiki.pid
